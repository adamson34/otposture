# VP-049: Floor Completeness

## Property Statement

If the gating set is empty, the final band equals the uncapped band and `floor_rule_triggered` is false — the floor rule never fires spuriously (Critical goals at PI/LI/FI/N-A never gate).

## Source Contract

- **BC:** BC-4.04.004 — Floor Rule
- **Postcondition/Invariant:** Postcondition 2, Invariant 3, EC-003

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_049() {
    let (catalog, answers, ov): (BoundedCatalog, BoundedAnswers, BoundedOverrides) = kani::any();
    kani::assume(catalog.is_valid());
    let r = score(&catalog, &answers, &ov);
    if r.floor.gaps.is_empty() {
        assert_eq!(r.band, r.band_uncapped);
        assert!(!r.floor.triggered);
    }
}
```
