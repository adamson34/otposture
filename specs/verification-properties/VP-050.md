# VP-050: Critical-Gap Exactness

## Property Statement

`critical_gaps[]` equals exactly the gating set: every member has effective tier Critical and level ∈ {NI, Unknown}; every goal satisfying that predicate is a member; each entry carries the correct tier source (default vs. override).

## Source Contract

- **BC:** BC-4.04.004 — Floor Rule
- **Postcondition/Invariant:** Postcondition 1, Invariant 2 (exhaustiveness)

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani | yes — ≤64 goals | set equality over symbolic goal array |
| proptest | proptest | no | unbounded confirmation incl. tier-source labels |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_050() {
    let (catalog, answers, ov): (BoundedCatalog, BoundedAnswers, BoundedOverrides) = kani::any();
    kani::assume(catalog.is_valid());
    let r = score(&catalog, &answers, &ov);
    for g in catalog.goals() {
        let gating = effective_tier(g, &ov) == Tier::Critical
            && matches!(answers.level(g), Level::NotImplemented | Level::Unknown);
        assert_eq!(gating, r.floor.gaps.contains(g.id()));
    }
}
```
