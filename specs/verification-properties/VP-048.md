# VP-048: Floor Soundness

## Property Statement

If the gating set — goals whose *effective* tier (after overrides) is Critical and whose level is Not Implemented or Unknown — is non-empty, the final band is ≤ the floor-cap band (positionally: the catalog's second-lowest band).

## Source Contract

- **BC:** BC-4.04.004 — Floor Rule
- **Postcondition/Invariant:** Postcondition 1, DI-003

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani | yes — ≤64 goals, ≤8 bands | symbolic answers, tiers, overrides |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_048() {
    let (catalog, answers, ov): (BoundedCatalog, BoundedAnswers, BoundedOverrides) = kani::any();
    kani::assume(catalog.is_valid());
    let r = score(&catalog, &answers, &ov);
    if !r.floor.gaps.is_empty() {
        assert!(r.band.index() <= 1); // second-lowest band, 0-indexed
        assert!(r.floor.triggered);
    }
}
```
