# VP-040: Percent Bounds

## Property Statement

For every validated catalog and every answer set, `score(...).percent` is either undefined (all goals N/A) or an exact rational in `[0, 1]`; it equals `1` iff every applicable goal is Fully Implemented.

## Source Contract

- **BC:** BC-4.04.001 — Weighted Percent Computation
- **Postcondition/Invariant:** Postcondition 1, Invariant 3 (bounds; FI-iff-1)

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani (pinned at provisioning) | yes — ≤64 goals, ≤8 functions, weights ≤10⁶ | full symbolic input space within bounds |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_040() {
    let catalog: BoundedCatalog = kani::any();
    kani::assume(catalog.is_valid()); // BC-1.01.002 rules as assumptions
    let answers: BoundedAnswers = kani::any();
    if let Some(p) = score(&catalog.into(), &answers.into(), &TierOverrides::none()).percent {
        assert!(p >= Rational::ZERO && p <= Rational::ONE);
        let all_fi = /* every applicable goal FI */;
        assert_eq!(p == Rational::ONE, all_fi);
    }
}
```
