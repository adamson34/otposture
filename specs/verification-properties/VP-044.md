# VP-044: Decomposition Identity

## Property Statement

The overall percent equals the weight-combined per-function sub-scores exactly: `overall = Σ(funcWeight_f × funcScore_f) / Σ(funcWeight_f)` over scoreable functions, where `funcWeight_f = Σ goal weights of f's applicable goals`. Exact rational equality — no tolerance.

## Source Contract

- **BC:** BC-4.04.002 — Per-Function Sub-Scores
- **Postcondition/Invariant:** Postcondition 2 (consistency between views)

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani | yes — ≤64 goals, ≤8 functions | full symbolic space within bounds |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_044() {
    let (catalog, answers): (BoundedCatalog, BoundedAnswers) = kani::any();
    kani::assume(catalog.is_valid());
    let r = score(&catalog, &answers, &none());
    if let Some(overall) = r.percent {
        let recombined = recombine(&r.functions); // Σ(w_f × s_f) / Σ(w_f)
        assert_eq!(overall, recombined); // exact rational equality
    }
}
```
