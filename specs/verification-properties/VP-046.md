# VP-046: Band Monotonicity

## Property Statement

The pre-floor band function is monotone: for percents `p1 ≤ p2` under the same validated band list, `band(p1) ≤ band(p2)` in band order.

## Source Contract

- **BC:** BC-4.04.003 — Maturity Band Derivation
- **Postcondition/Invariant:** Postcondition 1 + VP table

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani | yes — ≤8 bands | two symbolic percents, symbolic thresholds |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_046() {
    let bands: BoundedBands = kani::any();
    kani::assume(bands.is_valid()); // N>=2, strict increasing thresholds in (0,1)
    let (p1, p2): (Rational, Rational) = kani::any();
    kani::assume(in_unit(p1) && in_unit(p2) && p1 <= p2);
    assert!(band_index(&bands, p1) <= band_index(&bands, p2));
}
```
