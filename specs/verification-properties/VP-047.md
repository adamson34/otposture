# VP-047: Band Totality

## Property Statement

For every defined percent in `[0,1]` and every validated band list, exactly one band is assigned (no gaps, no overlaps), with each boundary value belonging to the higher band.

## Source Contract

- **BC:** BC-4.04.003 — Maturity Band Derivation
- **Postcondition/Invariant:** Postcondition 1 (closed-lower-bound rule)

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani | yes — ≤8 bands | symbolic percent incl. exact threshold values |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_047() {
    let bands: BoundedBands = kani::any();
    kani::assume(bands.is_valid());
    let p: Rational = kani::any();
    kani::assume(in_unit(p));
    let idx = band_index(&bands, p);
    assert!(idx < bands.len());                       // total
    // boundary-up: p == t_i implies idx == i+1
    if let Some(i) = bands.threshold_eq(p) { assert_eq!(idx, i + 1); }
}
```
