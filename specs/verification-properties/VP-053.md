# VP-053: Delta Reconciliation

## Property Statement

For any two comparable frozen assessments under one catalog: `Σ driver.points + composition_adjustment = percent_delta`, as exact rational equality. No unattributed movement can exist (DI-005).

## Source Contract

- **BC:** BC-4.04.006 — Delta Computation with Driver Attribution
- **Postcondition/Invariant:** Postcondition 2

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani | yes — ≤64 goals | symbolic pair of answer sets incl. N/A transitions |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_053() {
    let (catalog, from, to): (BoundedCatalog, BoundedAnswers, BoundedAnswers) = kani::any();
    kani::assume(catalog.is_valid());
    let d = delta(&catalog, &from.freeze(), &to.freeze());
    let driver_sum: Rational = d.drivers.iter().map(|dr| dr.points).sum();
    assert_eq!(driver_sum + d.composition_adjustment, d.percent_delta);
}
```
