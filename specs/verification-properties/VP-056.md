# VP-056: Comparability-Guard Soundness

## Property Statement

Any snapshot pair admitted by `check_comparability` shares an effective catalog digest — either both pin the identical `(id, version, digest)`, or the comparison uses back-cast values computed under the *to*-side catalog; and no admitted pair spans an unbridged Series Break.

## Source Contract

- **BC:** BC-4.04.007 — Comparability Guard
- **Postcondition/Invariant:** Postconditions 1–2, Invariant 1 (single chokepoint), DI-001

## Proof Method

| Method | Tool | Bounded? | Coverage |
|--------|------|----------|----------|
| kani | Kani | yes — ≤16 snapshots, ≤4 breaks | symbolic metadata + break intervals |
| proptest | proptest | no | randomized histories incl. bridged breaks |

## Proof Harness Skeleton

```rust
#[kani::proof]
fn verify_vp_056() {
    let (metas, breaks, a, b): (BoundedMetas, BoundedBreaks, usize, usize) = kani::any();
    kani::assume(a < metas.len() && b < metas.len());
    if let Ok(pair) = check_comparability(&metas[a], &metas[b], &breaks.as_slice()) {
        assert!(pair.effective_digest_equal());
        assert!(!breaks.unbridged_between(metas[a].seq, metas[b].seq));
    }
}
```
